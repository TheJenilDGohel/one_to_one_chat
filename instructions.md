One-to-one chat application using Flutter and Firebase.

Requirements:
• Create a screen to add users and display a list of users.
• When a user taps on a user from the list, it should open a chat screen for one-to-one messaging.
• Messages should be stored and synced using Firebase (Firestore).
• Implement real-time messaging so messages appear instantly.
• Add read/unread functionality for messages.
• (Optional) Show unread message count in the user list.

You are free to design the UI as you prefer. The main focus should be on functionality and Firebase integration.
